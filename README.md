# First time users
1.Download NPM

2.Get VSCODE (or any other editor)

3.Open the downloaded folder in VSCODE

4.Open up the terminal

5.Run Commands

Enter ``npm install``

Enter ``npm run start``

6.It will be launched on localhost (in your browser)
